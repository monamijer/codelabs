# 🚀 Journey Codelabs

Bienvenue dans mon laboratoire de code personnel. Ce dépôt est une collection d'exercices, de prototypes et de solutions accumulés tout au long de mon parcours d'apprentissage ("Journey"). 

L'objectif ici est double : documenter ma progression technique et maintenir un environnement de développement simple, léger et efficace.

## 📂 Structure du Projet

Le dépôt est organisé par modules thématiques pour faciliter la navigation :

* **`/html-css`** : Intégration moderne, Flexbox, Grid et sémantique.
* **`/javascript`** : Logique métier, manipulation du DOM et API Fetch.
* **`/backend-basics`** : Expérimentations avec Node.js, Express et bases de données.
* **`/projects`** : Petites applications complètes (ex: Weather App, Portfolio).

## 🛠 Environnement & Philosophie

Je privilégie des outils qui favorisent la concentration et la performance système :

* **Éditeur :** [Neovim](https://neovim.io/) (configuré en Lua pour la rapidité).
* **Shell :** [Fish Shell](https://fishshell.com/) pour une autocomplétion intelligente.
* **OS :** [Arch Linux](https://archlinux.org/) (Rolling release & KISS).
* **Approche :** **KISS** (*Keep It Simple, Stupid*). Je préfère comprendre comment les choses fonctionnent sous le capot avant d'ajouter des couches d'abstraction.

## ✨ Le Coin de la Sérendipité

*« On ne trouve pas ce que l'on cherche, on trouve ce que l'on ne cherchait pas. »* Voici une sélection de ressources pour sortir des sentiers battus et découvrir des concepts fascinants par hasard :

### 📚 Pour approfondir le Code
* **[Patterns.dev](https://www.patterns.dev/)** : Apprenez les design patterns JS et React de manière visuelle.
* **[The Modern JavaScript Tutorial](https://javascript.info/)** : De l'élémentaire aux concepts avancés comme les Proxies ou les Générateurs.
* **[A Guide to Flexbox](https://css-tricks.com/snippets/css/a-guide-to-flexbox/)** : La référence absolue de CSS-Tricks.

### 🧩 Pour la culture Tech & Terminal
* **[The Art of Command Line](https://github.com/jlevy/the-art-of-command-line)** : Une mine d'or pour devenir un magicien du terminal.
* **[Vim Adventures](https://vim-adventures.com/)** : Apprendre les raccourcis Vim à travers un jeu d'aventure.
* **[Low-Tech Magazine](https://solar.lowtechmagazine.com/fr/)** : Pour une réflexion sur l'informatique durable et frugale.

### 💡 Outils Utiles
* **[Can I Use](https://caniuse.com/)** : Pour vérifier le support des fonctionnalités web.
* **[Excalidraw](https://excalidraw.com/)** : Pour schématiser vos idées avec un rendu "dessiné main".
* **[Carbon](https://carbon.now.sh/)** : Pour créer de belles images de vos extraits de code.
